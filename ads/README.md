# Longgevia — Creatividades de lanzamiento (test)

Primeros banners para entender cómo montar la campaña. Editables: cada `.html` usa `_base.css` y las fotos reales de `../assets/`. Para regenerar un PNG, abre el HTML y haz captura al tamaño indicado (o pídeme el re-render).

## Formatos incluidos (`out/`)

| Archivo | Tamaño | Dónde |
|---|---|---|
| `ig-square.png` | 1080×1080 | Instagram feed, Facebook feed, carrusel |
| `ig-story.png` | 1080×1920 | Instagram/Facebook Stories y Reels |
| `fb-feed.png` | 1200×628 | Facebook feed enlace, IG horizontal |
| `x-post.png` | 1600×900 | X.com (Twitter) post / X Ads |

## Mensaje
- **Gancho:** "Recupérate hoy. Vive fuerte mañana."
- **Oferta:** suscripción · envío gratis el primer mes · desde 39 €/mes
- **CTA:** longgevia.com
- **Claims:** solo bienestar ("soporte nutricional", "rutina diaria"). Nada médico — cumple el criterio regulatorio del proyecto.
